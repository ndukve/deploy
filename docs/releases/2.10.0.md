# /docs/releases 2.10.0

**Release date:** 2026-06-14

## TL;DR

**CryptPad**, the end-to-end encrypted and open-source collaboration suite. It can be easily used to create documents, sheets, diagrams, boards, upload files.

Created and uploaded documents can also be shared with other CryptPad users.
CryptPad will currently directly login users that arrive with valid mTLS certificate and will create a persistent user in the CryptPad database with the users callsign.

Also includes some minor UI enhancements (https://github.com/pvarki/react-rasenmaeher-ui-v2/pull/54)

No breaking changes.

---

## Release Goals

- Give users ability to easily create documents, and share and upload files with other users.

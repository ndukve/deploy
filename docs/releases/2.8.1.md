# /docs/releases 2.8.1

**Release date:** 2026-04-27

## TL;DR

- Replaced list-based stream view with a card grid showing active streams with search and filtering
- New stream video page with iframe playback, download packages (Browser/VLC/TAK) and connection options dialog with credentials and protocol URLs
- New broadcast wizard with guided setup for OpenTAK ICU, GoPro, UAS Tool and advanced protocol configuration
- External guide links to docs.pvarki.fi for each broadcasting tool
- Full i18n support (EN/FI/SV) for all new UI text
- Incorporates SRT passphrase support from latest main (UAS Tool import, advanced links)

No breaking changes.

---

## Release Goals

- Improved stream discovery with visual card grid instead of plain list
- Step-by-step broadcast wizard reduces setup friction for new users
- Download packages let users quickly open streams in their preferred player
- Connection options dialog consolidates all stream access info in one place
